Master Mobile UX

  • Build Mobile first!!
  • Expand from the mobile, it will make it easier
  • Be There, Be Useful, Be Quick
  • Try to simplify the process
  • Use SVG/Webfonts for lower bandwidth
  • Minify CSS and JS files
  • Primary controls are well... Primary
  • put the menu on the bottom
  • Menu Icon is a lot better than the ≡ sometimes
  • KISS method - Keep it Stupid Simple
  • Make forms show only the data that they are willing to provide ie email or phone number based on checkboxes
  • Show passwords by default and add hide feature
  • Use number pad for certain fields
  • Use anchor tags to be able to have users click to call.
  • Stop using pop-ups
  • Make it sleek and focus on content first
